# DISCIPLINAS DO CURSO 
-Lógica de programação: é a base para criar algoritmos, sequencias lógicas, finitas e definidas de instruções, que resolvem problemas complexos e automatizam tarefas.
-Criatividade e inovação: A criatividade na engenharia de software é a geração de ideias originais para resolver problemas complexos, enquanto a inovação transforma essas ideias em soluções práticas, aplicativos ou sistemas funcionais, gerando valor e vantagem competitiva.
-Fundamentos para engs: Os fundamentos de engenharia de software englobam metodologias, processos e técnicas para desenvolver, operar e manter software com qualidade, dentro do prazo e orçamento.
-Programação e sistemas computacionais: Focada no ciclo de vida completo — análise, design, desenvolvimento, testes e manutenção — a área utiliza linguagens de programação, métodos ágeis e arquitetura de software para entregar soluções de alta qualidade.
